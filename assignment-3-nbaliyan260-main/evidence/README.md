# Evidence (by exercise)

Evidence from manual runs on the Tenstorrent machine, organized by exercise.

| Folder | Contents |
|--------|----------|
| **exercise1/** | Watcher debugging (no code deliverable): write-up, build-success screenshot |
| **exercise2/** | Ex2 (sender also computes): run log, profiler CSV, screenshots (run + profiler) |
| **exercise3/** | Ex3 (batched multicast): run log, profiler CSV, screenshots (run + profiler) |
| **exercise4/** | Ex4 (multicast matmul): run log, profiler CSV, screenshots (5×5, 2×2, profiler) |
| **lab2_baseline/** | Lab2 Ex2 (no multicast) — used for performance comparison with Ex4 |
| **performance/** | Performance comparison (Ex2 vs Ex3, Lab2 vs Ex4) |

Each exercise folder contains the relevant `.txt` logs, `.csv` profiler outputs, and `.png` screenshots for that exercise.
